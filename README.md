# Student Performance Prediction

## Project Overview
This project predicts student math performance using Machine Learning based on factors such as gender, race/ethnicity, parental level of education, lunch type, and test preparation course.

## Problem Statement
The objective is to understand how different factors affect student performance and build a machine learning model to predict math scores.

## Dataset
- Dataset: StudentsPerformance.csv
- Number of Records: 1000
- Number of Features: 8

## Technologies Used
- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn
- Scikit-learn

## Machine Learning Workflow
- Data Collection
- Data Cleaning
- Exploratory Data Analysis (EDA)
- Data Preprocessing
- Model Training
- Model Evaluation

## Models Used
- Linear Regression
- Random Forest Regressor

## Results
- Linear Regression R² Score: 0.8804
- Random Forest R² Score: 0.8488
